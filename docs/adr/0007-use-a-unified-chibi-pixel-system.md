# Use a unified chibi pixel visual system

Every Character Pet and Stand Pet will use one JoJo-inspired Chibi Pixel System built around roughly 2.5-head proportions, bold dark contours, limited character-specific palettes, and immediately recognizable silhouettes. This favors readability and identity consistency across the complete animated roster over manga-ink detail or 3D-toy rendering; assets will not trace individual manga panels or official illustrations, and Stands will remain visually related to Character Pets while using larger, more angular silhouettes.
