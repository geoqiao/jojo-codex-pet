# Lock one signature palette per pet

Each Codex Pet will use one Signature Palette across its complete v2 atlas, including all standard animation states and look directions. The Headline Roster will not include alternate palette editions or state-driven recoloring; this protects visual identity, keeps cross-row QA reliable, and prevents the 36-pet scope from multiplying, at the cost of omitting JoJo-style scene color shifts from the initial releases.
