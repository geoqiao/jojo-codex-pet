# Use one monorepo for pets and the gallery

The Fan Project will use the public repository slug `geoqiao/jojo-codex-pet` and keep Codex Pet packages, catalog metadata, gallery source, validation tooling, and documentation in that one repository. A single pull request can therefore add or change a pet together with its listing and pass one validation pipeline before deployment; this accepts a larger repository in exchange for avoiding cross-repository version drift and making future Community Submissions easier to review.
