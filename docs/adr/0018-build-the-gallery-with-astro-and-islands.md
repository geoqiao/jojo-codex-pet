# Build the gallery with Astro and focused interactive islands

The monorepo gallery will use Astro with TypeScript to pre-render stable home, Part, roster, installation, contribution, and pet-detail pages, while focused client islands handle catalog filtering, language selection, spoiler reveals, live sprite previews, and Showcase Effects. Its primary tabs will be Gallery, Parts, Roster, Install, and Contribute, with GitHub and the English/Chinese switch in the utility navigation; Contribute will explain that pull requests open only after Wave 1. This adapts the supplied community-gallery references to a JoJo-only catalog while preserving shareable URLs, metadata, and static deployment, accepting a mixed server-rendered and client-island component model instead of a single-page React application.
