# Use the GitHub repository as the source record

Project-authored pet metadata will use `https://github.com/geoqiao/jojo-codex-pet` as its canonical source record rather than preserving per-pet reference-image URLs or a separate provenance archive. Contributor identity and the repository history remain attributable, but third-party visual references will not be bundled as source material. This keeps the public package simple while accepting less granular provenance than a reference-by-reference ledger.
