# Provide cross-platform installers

Every published Codex Pet will be installable through four official entry points backed by the same package contract: an `Open in Codex` website action, a Bash command for macOS and Linux, a PowerShell command for Windows, and the `jojo-codex-pet` npm CLI invoked through `npx`. Each path will resolve the requested Catalog Pet ID, verify `pet.json` and the v2 spritesheet before replacement, and install only the two runtime files under the active Codex home. The primary action installs only the selected pet; installing its Stand Pair or complete Part Collection remains a separate explicit action. This accepts maintaining multiple thin platform wrappers in exchange for a predictable cross-platform experience without surprise bulk installation.
