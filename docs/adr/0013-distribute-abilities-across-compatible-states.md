# Distribute abilities across compatible animation states

A Core Ability Cue will appear through multiple Ability Beats across semantically compatible standard animation states instead of being confined to the active-work row. There is no numeric quota: any standard state may receive an Ability Beat when it strengthens that state's meaning and passes pet-safe visual QA, while the 16 look directions remain effect-free. This gives each power a broader expressive arc while accepting additional and deliberately variable per-pet design and QA work.
