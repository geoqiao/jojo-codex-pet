# Open JoJo pull-request submissions after Wave 1

The Fan Project will keep pet creation maintainer-led through the first Release Wave, then open a Contribution Gate for JoJo-only Expansion Pets submitted through GitHub pull requests. Wave 1 establishes the package schema, visual baseline, automated checks, and review examples before outside contributions arrive; the website will not initially require accounts, direct uploads, likes, or a moderation backend, accepting delayed community participation in exchange for a controlled standard.
