# Publish immediately on the temporary domain

The GitHub repository will be public from its creation, and the first usable Gallery release will be indexable at `jojo-preview.easytry.shop` without a staging-only or `noindex` phase. Until a dedicated domain is acquired, this subdomain is the canonical production site rather than a disposable preview; a later domain migration must preserve routes and redirect the temporary host. This favors immediate public iteration and accepts that early versions and fan-project risk will be visible from launch.
